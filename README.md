# Gibrex

## Howto

1. Clone the repo
``` bash
 $ git clone git@gitlab.tracto.org:syahrun/agibrex.git
```

2. Install the npm dependencies
``` bash
 $ npm install
```

3. Init the database
``` bash
 $ ./ace migration:run
```

4. Serve the pages, and continue developing
``` bash
 $ npm run serve:dev
```
